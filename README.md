# Homebrew Tap for xctree

This is the official Homebrew tap for [xctree](https://github.com/ldomaradzki/xctree), a command-line tool for extracting iOS Simulator accessibility trees.

## Installation

```bash
brew tap ldomaradzki/xctree
brew install xctree
```

## Install HEAD (Development Version)

To install the latest development version directly from the main repository:

```bash
brew install --HEAD ldomaradzki/xctree/xctree
```

## Updating

```bash
brew update
brew upgrade xctree
```

## About

xctree is a Swift-based CLI tool that provides accessibility tree inspection for iOS Simulator apps, similar to Xcode's Accessibility Inspector but designed for terminal use.

For more information, see the [main repository](https://github.com/ldomaradzki/xctree).
